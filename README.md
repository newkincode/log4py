## Menual
### 1. Install
```bash
$ wget https://github.com/newkincode/log4py/blob/main/log4py.py
```
### 2. Import
```python
from log4py import Logger
```
### 3. Use
```python
logger = Logger()
logger.info("Hello World!")
```
### 4. Result
```
0000-00-00 00-00-00 - INFO - Hello World!
```
